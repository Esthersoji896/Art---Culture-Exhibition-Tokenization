# Art & Culture Exhibition Tokenization

A decentralized platform for tokenizing museum entries and gallery pieces as NFTs with historical metadata.

## 🌟 Features

- Mint NFTs with detailed artwork metadata
- Market functionality for buying and selling pieces
- Authenticity verification system
- Comprehensive metadata storage
- Secure ownership transfer

## 🛠 Technical Stack

- Clarity Smart Contracts
- Stacks Blockchain
- SIP-009 NFT Standard
- Clarinet Testing Framework

## 🚀 Getting Started

1. Install Clarinet
```bash
curl -L https://github.com/hirosystems/clarinet/releases/download/v1.0.0/clarinet-linux-x64.tar.gz | tar xz
```

2. Initialize project
```bash
clarinet new art-culture-exhibition
```

3. Run tests
```bash
clarinet test
```
